    isAccessibleForFree





[more...]


 Canonical URL: https://schema.org/isAccessibleForFree 

Check for open issues.




A flag to signal that the item, event, or place is accessible for free.
 

Values expected to be one of these types


Boolean



Used on these types


CreativeWork

Event

Place





Supersedes

free
